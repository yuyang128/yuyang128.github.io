What’s an HTTP Proxy什么是 HTTP 代理

A proxy is some forwarder application that connects your HTTP client to backend resources. HTTP clients can be browsers or applications like curl, SOAP UI, Postman, etc. Usually, these proxies are used for routing and getting internet access when there is no direct connection to the internet from the client itself. HTTP proxies are therefore also ideal when you are testing your application. You can always use the proxy log records to see what was actually sent from client to server. So you can check the request and response headers and the XML, JSON, or other payloads.代理是一种将你的 HTTP 客户端连接到后端资源的转发应用。HTTP 客户端可以是浏览器或应用程序，如 curl、SOAP UI、Postman 等。通常，当客户端本身没有直接互联网连接时，这些代理用于路由和获取互联网访问。因此，HTTP 代理在测试应用时同样理想。你总可以用代理日志记录查看客户端到服务器实际发送了哪些内容。所以你可以检查请求和响应头，以及 XML、JSON 或其他负载。

HTTP Proxies receive requests from a client and relay them. They also typically record them. They act as a man-in-the-middle. It even works fine with or without HTTPS as long as your client or browser trusts the certificate of the HTTP Proxy.HTTP 代理接收客户端的请求并中继。他们通常也会录制这些视频。他们充当中间人的角色。只要你的客户端或浏览器信任 HTTP 代理的证书，它也能正常运行。
